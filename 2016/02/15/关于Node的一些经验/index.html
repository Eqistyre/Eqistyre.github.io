    • CORS

    CORS(Cross-Origin Resource Sharing)跨域资源分享。

    跨域资源共享,是一种允许当前域（domain）的资源（比如html/js/web service）被其他域（domain）的脚本请求访问的机制，通常由于同域安全策略（the same-origin security policy）浏览器会禁止这种跨域请求。

    也就是说对于一个api如果目标域没有处理的话，我们将无法获取目标域的资源。

    A JSONP response is just a script. That’s it. If a response is not a script, it’s not a JSONP response. When you request a resource from a server, the server decides whether or not it will serve you a script; there’s nothing more you can do about it, unless you can reprogram the server. In this case, [{JSON1}, {JSON2} ..] is not a script (or at least, not a script that does anything). Therefore, it’s not JSONP. Unless you control the server, you can’t make the server start serving you script files like doStuff([…]) instead of JSON files like […]. – apsillers Dec 2 ‘14 at 19:48

    也就是说JSONP回应东西仅仅只是script。如果返回的不是script的话，那么它也就不是JSONP回应了。当你向服务器发出资源请求的时候，服务器将决定是否提供给你一个script，对此你无能为力，除非你能够对服务器进行编程。
