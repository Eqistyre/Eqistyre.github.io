JSX

在javaScript代码里写着XML格式的代码称为JSX; 他没有修改javaScript的语义。

要渲染HTML标签，只需在JSX中使用小写字母开头的标签名。

要渲染React组件，只需创建一个大写字母开头的本地变量。

React的JSX里约定分别使用首字母的大、小写来区分本地组件的类和HTML标签。

转换

JSX把类XML的语法转换成纯粹JavaScript，XML元素、属性和子节点被转换成React.createElement的参数。
